# MyFootball

## 해외 축구 일정 및 순위 확인

EPL과 네이버 해외축구 페이지의 UI를 참고하여 만든 해외 축구 리그 일정 및 순위표 보기 웹페이지입니다.

## API

https://www.football-data.org

## 기술

React + NestJS
CSS Material ui

API Control : Tanstack-Query
ErrorAlertState: Recoil
